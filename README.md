# Unreal Third Person Swords Game

Third-person sword-fighting project built with Unreal Engine.

## Project
- Engine: Unreal Engine 5.5 (from `A1U/A1U.uproject`)
- Project file: `A1U/A1U.uproject`

## Getting Started
1. Install Unreal Engine 5.5.
2. Open `A1U/A1U.uproject` in the Unreal Editor.
3. Let the editor generate project files if prompted.
4. Build and run from the editor.

## Repo Notes
- Large Unreal assets are tracked with Git LFS (`*.uasset`, `*.umap`).
- Generated folders are ignored (`Binaries/`, `Intermediate/`, `DerivedDataCache/`, `Saved/`, `.vs/`).

## Structure
- `A1U/Content/` Game assets (maps, meshes, materials, blueprints).
- `A1U/Config/` Project configuration.
- `A1U/Source/` C++ source.
